** Please note: Port numbers given in this text are examples only, 
   you can use other numbers too!

COMPILE instructions:
     After uploading all files to a directory on the OSU server, navigate
     to the project1 directory: makefile
	 

	Please ensure included makefile is in the current directory.
	Type 'make' without the single quotes to compile.

EXECUTE:
     On Host A (for example, "flip1" server) enter:
          java chatserve <Port Number>
     On Host B (for example, "flip2" server) enter:
          python chatclient.py <Host A> <Host A's Port Number>

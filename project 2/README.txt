**Running ftclient.py**

Give it executable permissions: chmod +x ftclient.py

Run the executable ./ftclient.py <server_port> -l <data_port>

OR

./ftclient.py <server_port> -g <file_to_get> <data_port>



**ADDED ISSUE**
If complier gives warning about the interpretter, then the program is meant for flip.

run it with command on flip: python ftclient.py <server_port> -l <data_port>

OR

python ftclient.py <server_port> -g <file_to_get> <data_port>


**** MAKE SURE THAT THE EXECUTABLES ARE IN DIFFERENT DIRECTORIES ****

Program was tested on flip2.engr.oregonstate.edu and flip3.engr.oregonstate.edu
